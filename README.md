# 🏋️ Workout API

## 📌 Project Description

The Workout API is a RESTful backend service built with Flask, SQLAlchemy, and Marshmallow. It allows users to manage workouts and exercises, and track how exercises are performed within each workout using sets, reps, and duration.

Each workout can contain multiple reusable exercises, making the system flexible for personal trainers and fitness tracking applications.
